## install packages

https://pub.dev/

## run

```
flutter pub get
flutter run -d chrome
flutter run -d windows
```

check for problems
```
flutter doctor
```

## manage flutter

upgrade flutter:
```
flutter upgrade
```

upgrade packages (list outdated: `flutter pub outdated`):
```
flutter pub upgrade
```
